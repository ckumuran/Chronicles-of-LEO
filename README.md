# BRIXIT

BRIXIT is a Minecraft-inspired sandbox survival game where every block is made of colorful LEGO-style plastic bricks.

Mine blocks.
Build structures.
Explore infinite terrain.
Survive in a world built entirely from studs.

---

## Features

- Infinite procedural terrain
- LEGO-style plastic materials
- Block placing and breaking
- Smooth first-person controls
- Dynamic lighting
- Chunk generation
- Inventory system
- Stylized voxel graphics
- Brick particle effects

---

## Tech Stack

- React
- Vite
- Three.js
- React Three Fiber
- React Three Drei
- Rapier Physics
